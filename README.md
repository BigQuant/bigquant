# BigQuant SDK

BigQuant Financial Quantitative Toolbox — 金融量化工具箱

`bigquant` 是 [BigQuant](https://bigquant.com) 的开源 Python SDK 和 CLI 工具，为用户和 AI 智能体提供金融量化分析、策略回测和交易的编程接口。

> **Under Active Development** — 代码正在积极开发中，即将发布首个版本。欢迎关注、Star 和提交反馈。
